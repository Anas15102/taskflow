# Demo Video Script (2-5 minutes)

## Introduction (15 seconds)
"Hi, I'm demonstrating TaskFlow, a full-stack team task management application with role-based access control."

## 1. User Authentication (30 seconds)
- Show the login page
- Click "Sign up"
- Fill in registration form (name, email, password, select role)
- Successfully register and auto-login
- Show the dashboard

## 2. Dashboard Overview (30 seconds)
- Point out the statistics cards (Total Tasks, To Do, In Progress, Completed, Overdue)
- Show the navigation (Dashboard, Projects)
- Explain the clean, modern UI

## 3. Create Project (45 seconds)
- Click "Projects" in navigation
- Click "New Project" button
- Fill in project details:
  - Name: "Website Redesign"
  - Description: "Redesign company website with modern UI"
- Create the project
- Show the project card with member count

## 4. Project Management (60 seconds)
- Click on the project to open details
- Show team members section
- Click "Add Member" (if admin)
- Search for a user and add them
- Show the member list with roles (admin/member badges)

## 5. Task Management (90 seconds)
- Click "New Task" button
- Create first task:
  - Title: "Design homepage mockup"
  - Description: "Create modern homepage design"
  - Assign to a team member
  - Priority: High
  - Due date: Set a date
- Create second task:
  - Title: "Setup development environment"
  - Priority: Medium
  - Status: In Progress
- Create third task:
  - Title: "Research competitor websites"
  - Priority: Low
  - Status: Completed

## 6. Kanban Board (30 seconds)
- Show the three columns: To Do, In Progress, Completed
- Drag or use dropdown to change task status
- Show how tasks move between columns
- Point out priority badges and due dates
- Show overdue task highlighting (if any)

## 7. Role-Based Access (20 seconds)
- Explain admin can:
  - Add/remove members
  - Delete tasks
  - Manage project settings
- Members can:
  - Create tasks
  - Update task status
  - View all project information

## 8. Dashboard Return (20 seconds)
- Go back to Dashboard
- Show updated statistics
- Point out recent tasks list
- Show overdue alerts (if any)

## Conclusion (20 seconds)
"TaskFlow provides complete project and task management with authentication, role-based access, and an intuitive interface. The app is built with React, Node.js, Express, and MongoDB, and is fully deployed on Railway."

## Technical Highlights to Mention
- Full-stack MERN application
- JWT authentication
- MongoDB with proper relationships
- Role-based access control (Admin/Member)
- RESTful API design
- Responsive design
- Production deployment on Railway

## Recording Tips
1. Use a clean browser window (incognito mode)
2. Close unnecessary tabs
3. Zoom in browser to 110-125% for better visibility
4. Speak clearly and at a moderate pace
5. Show smooth transitions between features
6. Have test data ready beforehand
7. Keep it under 5 minutes
8. End with showing the live URL
